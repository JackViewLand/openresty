# openresty
openresty 是基於```nginx在加上lua腳本```的web server，比起單純的nginx，多了lua腳本的擴展性。

這個專案目前是一個簡單的web server，用於我拿來進行源站測試。

* clone 專案
  ```
  git clone https://github.com/JackViewLand/openresty.git
  ```

* 執行docker-compose
  ```
  docker-compose up -d
  ```
